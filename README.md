# BmLerp

Lerp for Unity Object

Lerp函数的一些应用，无需写代码完成拼接。

目前有Transform的 位移，旋转，缩放

UI中Graphics的Color

可以将Animator放入组里和其它的属性组合

通过BmLerpGroup可以批量处理，支持Group嵌套

通过BmLerpAnimation来播放动画

增加了对TimeLine的支持
